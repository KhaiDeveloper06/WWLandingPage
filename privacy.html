Privacy Policy

Last Updated: July 2026

1. Overview

The Wolvesville Tactical Overlay & Analytics Tracker ("we," "our," or "the Extension") is committed to maintaining strict user privacy. This Extension operates as a client-side utility helper to assist players with manual notes, chat summaries, and vote history tracking during active matches on Wolvesville.com.

2. Core Match Data Collection & Transmission

We believe your gameplay data belongs to you. The core features of the Extension adhere to a strict local-only protocol:

  • No Server Communications for Match Data: The Extension does not possess any remote server infrastructure for gameplay analytics. It does not transmit your match logs or chat data over the network.
  • No Personal Game Data Scraping: We do not read, store, or log user accounts, login credentials, or game authentication tokens.
  • Volatile Match Logging: Your chat analytics charts and voting matrices are compiled entirely inside temporary runtime variables. They disappear when you close or refresh the match tab.

3. License Verification & Third-Party Integrations

To support the ongoing development of the Extension, we may offer optional premium features and display integrations:

  • Separation of License and Gameplay Data: If you choose to upgrade to a premium tier, transaction data is processed entirely by our secure third-party Merchant of Record (e.g., LemonSqueezy, Stripe). The Extension only locally stores a cryptographic license validation token to verify your premium status. It does not read, process, or store your payment details.
  • Ad & Affiliate Networks: The free tier of the Extension may include non-intrusive affiliate links or display banners provided by third-party gaming networks. These networks may use standard browser cookies to serve contextually relevant content, but they do not have access to your in-game chat or vote analytics.

4. Chrome Extension Permissions Declared

To execute its overlay functionalities seamlessly under Manifest V3 parameters, the Extension requests the following browser execution pathways:

chrome.storage.local

Utilized strictly to save your preferred localized language settings, UI theme configurations, and premium license validation tokens persistently across sessions.

activeTab / Host Permissions

Restricted strictly to the https://*.wolvesville.com/* origin space. This allows the script engine execution access to parse active open DOM tree frames to cleanly render the tactical control node and process visible match layouts locally.

5. Developer Policy Compliance Guarantee

The Extension strictly complies with the Google Chrome Web Store Single Purpose Policy, User Data Privacy Guidelines, and Developer Program Rules. The Extension operates purely via passive HTML DOM element parsing. It does not intercept, read, modify, or transmit WebSocket network payloads, ensuring absolute integrity alongside official game fair-play regulations.
