Privacy Policy

Last Updated: July 2026

1. Overview

The Wolvesville Tactical Overlay & Analytics Tracker ("we," "our," or "the Extension") is committed to maintaining strict user privacy. This Extension operates entirely as a client-side utility helper to assist players with manual notes, chat summaries, and vote history tracking during active matches on Wolvesville.com.

2. Zero Data Collection & Transmission

We believe your data belongs to you. The Extension adheres to a strict zero-collection protocol:

  • No Server Communications: The Extension does not possess any remote server infrastructure or analytics backend endpoints. It cannot transmit information over the network.
  • No Personal Data Scraping: We do not read, store, or log user accounts, login credentials, emails, authentication tokens, or payment details.
  • No Match Logs History Harvesting: Your chat analytics charts and voting matrices are compiled entirely inside temporary volatile runtime variables. They disappear when you close or refresh the match tab.

3. Chrome Extension Permissions Declared

To execute its overlay functionalities seamlessly under Manifest V3 parameters, the Extension requests the following browser execution pathways:

chrome.storage.local

Utilized strictly to save your preferred localized language settings (wovLang) and UI theme layout configuration options (wovDarkMode) persistently across sessions.

activeTab / Host Permissions

Restricted strictly to the https://*.wolvesville.com/* origin space. This allows the script engine execution access to parse active open DOM tree frames to cleanly render the tactical control node and process visible match layouts locally.

4. Security Infrastructure

Because all operational tracking mechanics compile within your browser's local sandbox memory infrastructure, the security of your tactical metrics is tethered entirely to your personal device architecture. No outside malicious third parties can access your logged analytics because the data never crosses a network bridge.

5. Developer Policy Compliance Guarantee

The Extension strictly complies with the Google Chrome Web Store Single Purpose Policy, User Data Privacy Guidelines, and Developer Program Rules. It does not automate inputs, read underlying WebSocket binaries, or simulate user interactions, ensuring absolute integrity alongside official fair-play regulations.

6. Contact & Support

For code architectural clarifications, feature requests, or direct security infrastructure inquiries, please open an issue tracking card via our official community development pipeline or open documentation avenues listed on our main landing interface.
